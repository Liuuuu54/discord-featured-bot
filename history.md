# 更新历史

## v2.1.0

- **架构解耦**: 完成六阶段重构，拆分启动流程、Bot client、精选指令层、书单子系统、互动 View 与共用 utils。
- **兼容入口**: `bot.py` 保留为旧部署兼容入口，实际启动流程迁移到 `app/main.py`。
- **精选模块拆分**: 精选 Cog 保留在 `app/features/featured_system.py`，View/Modal 拆分到 `feature_actions.py`、`record_views.py`、`stats_views.py`、`appreciator_views.py`。
- **书单模块拆分**: 书单子系统迁移到 `app/booklist/`，并拆分为 commands、modals、views、formatting、constants；根目录 `booklist_system.py` 保留兼容导出。
- **测试补强**: 新增 `tests/test_database_manager.py`，覆盖 SQLite 精选、排行、书单、书单帖守门等核心行为。
- **文档拆分**: README 精简为项目入口，指令说明移至 `command.md`，更新历史移至 `history.md`。

## v2.0.2

- **书单管理面板优化**: 按钮顺序重排，提升高频操作效率。
- **URL 加帖入口**: 面板新增“加帖子”按钮，支持 `书单ID + 帖URL + 评论` 直接添加。
- **展示格式优化**: 书单管理与公开书单统一为 `ID / 标题 / 连结 / 评价` 四栏显示。
- **全服书单管理**: 新增 `/书单 全服书单列表`，支持白名单设置与批量解绑。
- **误绑防护**: 绑定书单帖时强制校验“操作用户必须是该帖楼主”。

## v2.0.1

- **书单帖守门**: 绑定书单帖后，仅绑定者（楼主）可在该帖发言，其他用户留言会被 Bot 自动删除（反应不受影响）。
- **解绑即失效**: 清除书单帖连结后，守门规则立即取消。
- **权限要求**: 守门依赖 Bot 在对应频道拥有 `Manage Messages` 权限。

## v2.0.0

- **书单系统**: 新增 `/书单 添加至书单`、`/书单 管理书单`、`/书单 公开书单` 三个核心命令。
- **独立模块化**: 书单逻辑拆分到独立脚本，避免与主业务耦合。
- **书单规则落地**: 每用户 10 张书单、每张最多 20 帖、同书单帖子去重。
- **跨服收藏**: 用户可在有 Bot 的不同服务器中收藏帖子到同一账号书单。
- **书单帖跳转**: 新增书单帖 URL 绑定，并在 `/留言 精选记录` 中展示。
- **公开书单分页**: 公开书单每页 5 帖，所有用户可翻页。
- **重启持久化**: 公开书单翻页按钮支持重启后恢复。
- **索引自动清理**: 删除公开书单消息时自动删除最小索引，避免 DB 膨胀。
- **鉴赏条件扩展**: 鉴赏家申请新增“已绑定书单帖连结”作为可选达标条件之一。
- **公开限制增强**: `/书单 公开书单` 限定论坛帖且需楼主执行。

## v1.5.0

- **右键菜单功能**: 新增右键菜单精选功能，支持直接右键点击留言进行精选、取消精选和查看统计。
- **交互式表单**: 右键精选时弹出交互表单，方便输入精选原因。
- **功能对等**: 右键菜单功能与斜杠命令完全对等，包括权限检查、通知效果。
- **统计查看**: 新增右键查看统计功能，可直接查看留言作者的精选统计。
- **积分系统完全移除**: 移除积分系统，简化管理，只保留精选记录和引荐统计。

## v1.4.2

- **配置系统重构**: 大幅简化配置管理，将敏感信息（`DISCORD_TOKEN`）与环境配置分离。
- **文档更新**: 更新 `CONFIG_GUIDE.md`、`env_example.txt` 等文档，简化配置说明。
- **配置检查工具**: 新增 `config_check.py` 工具，方便用户验证配置正确性。
- **Docker 配置优化**: 简化 `docker-compose.yml` 环境变量配置，减少重复设置。

## v1.4.1

- **全服精选列表管理**: 新增 `/留言 全服精选列表` 命令，管理组可查看全服所有精选留言列表。
- **高级排序**: 支持按时间排序（默认）和按赞数排序两种模式。
- **时间范围筛选**: 支持起始日期和结束日期参数，灵活查询特定时间段。

## v1.4.0

- **增强排行榜**: `/留言 总排行` 命令支持积分排行和引荐人数排行两种模式切换。
- **引荐排行**: 新增引荐人数排行榜，方便调查作者使用 bot 的情况。
- **时间范围支持**: `/留言 总排行` 命令新增时间范围参数。
- **功能整合**: 移除独立的 `/月排行` 命令，功能整合到 `/留言 总排行` 中。
- **帖子统计排序**: `/留言 帖子统计` 命令新增赞数排序功能。
- **积分记录切换**: `/积分` 命令新增被精选记录和引荐记录切换功能。
- **界面优化**: 引荐相关按钮统一使用绿色样式，提升视觉一致性。
