# 群组数据提取工具使用指南

## 概述

`guild_data_extractor.py` 是一个专门用于从数据库中提取指定群组所有数据的工具。支持导出为JSON和CSV格式，方便数据备份、迁移和分析。

## 功能特点

- 🔍 **精确提取**: 只提取指定群组的数据，确保数据隔离
- 📊 **完整数据**: 包含精選记录等所有数据
- 📁 **多格式导出**: 支持JSON、CSV和独立数据库格式
- 📋 **数据摘要**: 提供详细的数据统计信息
- 🕒 **时间戳**: 自动添加提取时间戳，便于版本管理
- 🗄️ **独立数据库**: 可创建包含指定群组数据的完整数据库文件

## 使用方法

### 1. 查看所有群组

```bash
python guild_data_extractor.py list
```

输出示例：
```
🔧 群组数据提取工具
==================================================
✅ 成功连接到数据库: data/featured_messages.db
📋 数据库中共有 2 个群组:
  🏠 群组 123456789: 15 用户, 12 精選
  🏠 群组 987654321: 8 用户, 7 精選
🔌 数据库连接已关闭
```

### 2. 提取指定群组数据

```bash
# 提取群组数据（默认导出JSON和CSV）
python guild_data_extractor.py 123456789

# 只导出JSON格式
python guild_data_extractor.py 123456789 json

# 只导出CSV格式
python guild_data_extractor.py 123456789 csv

# 只创建独立数据库
python guild_data_extractor.py 123456789 db
```

### 3. 提取所有群组数据

```bash
python guild_data_extractor.py all
```

## 输出文件

### JSON格式
- `guild_123456789_20250127_143022.json` - 包含所有数据的JSON文件

### CSV格式
- `guild_123456789_20250127_143022_featured_messages.csv` - 精選记录数据

### 独立数据库格式
- `guild_123456789_20250127_143022.db` - 完整的SQLite数据库文件，包含指定群组的所有数据

## 数据字段说明

### 精選记录 (featured_messages)
- `id`: 记录ID
- `thread_id`: 帖子ID
- `message_id`: 留言ID
- `author_id`: 留言作者ID
- `author_name`: 留言作者名
- `featured_by_id`: 精選者ID
- `featured_by_name`: 精選者名
- `featured_at`: 精選时间
- `reason`: 精選原因
- `bot_message_id`: 机器人精選通知消息ID

## 注意事项

1. **数据完整性**: 提取的数据包含群组的所有相关记录
2. **时间戳**: 文件名包含时间戳，避免覆盖之前的数据
3. **权限要求**: 确保对数据库文件有读取权限
4. **存储空间**: 大量数据可能占用较多存储空间
5. **数据隔离**: 每个群组的数据完全独立，不会混淆

## 错误处理

### 常见错误及解决方案

1. **群组不存在**
   ```
   ❌ 群组 123456789 在数据库中不存在
   可用的群组: [987654321, 111222333]
   ```
   解决方案: 使用 `list` 命令查看可用的群组

2. **数据库连接失败**
   ```
   ❌ 连接数据库失败: [Errno 2] No such file or directory
   ```
   解决方案: 检查数据库文件路径是否正确

3. **权限不足**
   ```
   ❌ 保存JSON文件失败: [Errno 13] Permission denied
   ```
   解决方案: 检查当前目录的写入权限
