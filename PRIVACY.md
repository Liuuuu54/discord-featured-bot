# 隱私權政策 / Privacy Policy

*(For Discord review purposes: this document is written in Traditional Chinese for the bot's actual users; an English summary is provided at the top.)*

**English summary:** This bot stores, on its own self-hosted server (not shared with third parties beyond the companion web page described below): Discord user IDs and username snapshots, guild/channel/message/thread IDs, an optional admin-entered "reason" note, and user-authored booklist entries. It does not persistently store the full text of featured messages. Data can be deleted on request — contact wizozc12883@gmail.com.

---

## 我們蒐集什麼資料

本機器人(以下稱「本 Bot」)在提供「留言精選」與「書單」功能時，會將以下資料儲存在我們自行架設、管理的伺服器（VPS）上：

- **Discord 識別碼**：使用者 ID、伺服器（Guild）ID、頻道／討論串 ID、訊息 ID
- **使用者名稱快照**：留言作者、精選操作者在被記錄當下的顯示名稱
- **精選附註**：管理員／樓主為精選留言填寫的選填備註文字
- **書單內容**：使用者自行輸入的書單項目（書名、備註、討論串連結等）與書單公開設定
- **伺服器功能開關設定**：各伺服器管理員設定的功能開關（例如網頁接管、書單守門、歡迎頻道）

本 Bot **不會**永久儲存留言的完整原文——讀取留言內容僅用於當下的品質檢查（例如長度、是否為純表情），檢查後不寫入資料庫。

## 我們如何使用資料

- 提供「留言精選」「排行榜」「書單」「新成員歡迎」等 Bot 核心功能
- 依伺服器管理員設定，將書單資料與我們的網頁版（[odysseia-forum-webpage.pages.dev](https://odysseia-forum-webpage.pages.dev)）進行同步發佈（僅限伺服器管理員主動開啟此功能時）

## 資料分享

除上述網頁版同步功能外，我們不會將您的資料出售或分享給任何第三方。

## 資料保存與刪除

資料會保留至您要求刪除，或該筆記錄所屬的伺服器不再使用本 Bot 為止。如需刪除您的個人資料，請透過下方聯絡方式與我們聯繫。
